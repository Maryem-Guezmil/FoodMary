# FoodMary App

A website for selling delicious food online

## Running Locally

1. clone this repo
2. cd frontend
3. npm i --force
4. ng serve -o
   
Then open another new terminal 
1.  cd backend
2. npm i --force
3. npm start

## Tech Stack

1. Angular
2. Node.jS/express.jS
3. MongoDB/Mongoose
4. Heroku (hosting)
